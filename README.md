# ZIZA Shop - E-commerce Website

ZIZA Shop is a modern, responsive e-commerce front-end application built with React. It allows users to browse, search, and purchase the latest smartphones and laptops.

## Features

- **Product Catalog:** Browse products neatly organized by categories like "Smartphones" and "Laptops".
- **Powerful Search:** Quickly find any product using the search bar.
- **Product Details:** View detailed information, including descriptions, pricing, and ratings for each product.
- **Shopping Cart:** Add products to a cart and manage quantities or remove items before checkout.
- **Responsive Design:** A seamless user experience on both desktop and mobile devices.
- **Interactive UI:** Smooth carousels and sliders for product viewing, powered by Swiper.js.
- **User-Friendly Notifications:** Clean and modern alerts for cart interactions using SweetAlert2.
- **Contact Form:** A functional contact form for user inquiries.

## Technologies Used

- **Front-End:**
  - [React](httpss://reactjs.org/)
  - [Vite](httpss://vitejs.dev/)
  - [Tailwind CSS](httpss://tailwindcss.com/)
- **Routing:**
  - [React Router](httpss://reactrouter.com/)
- **UI Components & Libraries:**
  - [Swiper.js](httpss://swiperjs.com/) for carousels.
  - [React Awesome Reveal](httpss://github.com/morellodev/react-awesome-reveal) for scroll animations.
  - [SweetAlert2](httpss://sweetalert2.github.io/) for pop-up notifications.
  - [React Icons](httpss://react-icons.github.io/react-icons/)
- **API:**
  - [DummyJSON](httpss://dummyjson.com/) for product data.
